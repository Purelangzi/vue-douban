# vue-douban
仿豆瓣移动端开发，使用vue-cli4.x脚手架搭建项目

技术栈vue2.6 + vue-router + vuex + axios + flex + ES6/7 + scss +mint

使用rem实现自适应

字体图标库：Font Awesome
